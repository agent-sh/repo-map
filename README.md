# repo-map

AST-based repository map generator using ast-grep for symbol and import extraction with incremental updates

## Installation

```bash
# Claude Code
claude mcp add-plugin agent-sh/repo-map

# Or install from marketplace
agentsys install repo-map
```

## Usage

```
/repo-map
```
